发音为 `/tɑːˈkuːmi/`，Takumi 是一个能自动化开发任务并提供智能建议的 AI 编程助手。

![Takumi 演示](https://cdn.jsdelivr.net/gh/sorrycc-bot/image-2025-04@main/uPic/takumi-20250618-1.gif)

## 功能特性

- 代码生成与优化
- 上下文感知建议
- 支持多 LLM 提供商
- 集成 IDE 工作流

## 快速开始

环境要求: Node.js 18+

```bash
npm install -g takumi
export OPENAI_API_KEY="您的密钥"  # 或其他提供商
takumi
> help  # 查看命令
```
